# nicklemmon.com

![Build and test](https://github.com/nicklemmon/nicklemmon-blog-next/workflows/Build%20and%20test/badge.svg)

## Development

```
npm run dev
```

### Husky

If git hooks aren't running, try:

```bash
chmod ug+x .husky/*
chmod ug+x .git/hooks/*
```

## Production

```
npm run build
```

### Running a Production Server

```
npm start
```

## Testing

```
npm run test:e2e
```
